# Setup

`yarn`

# Run tests

`yarn test {relative path}` - this will run `jest {relative path`

or

`yarn test --bail` - stop as soon as one suite fails

or

`yarn test:watch` - this will run `jest --watchAll`

# Run without tests

`yarn start {filepath}` - this will run `node {relative path}`
