# Setup

`yarn`

# Run tests

`yarn test` - this will run `jest`
 
or

`yarn test --bail` - stop as soon as one suite fails

or

`yarn test:watch` - this will run `jest --watchAll`

# Run without tests

`yarn start katas/{filepath}` - this will run node src/katas{filepath}
